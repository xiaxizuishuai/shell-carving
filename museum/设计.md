**3. 核心展示区 (两列布局 Flex/Grid)**
在一个宽容器中并排两个毛玻璃卡片容器。

* **左侧卡片：Featured Collection（重点：蓝绿渐变相框）**
    * 标题居中。
    * 包含两个并排的项目（Queen Conch 和 Nautilus）。
    * **相框定制 CSS 需求**：图片的外部包裹一个正方形的相框。这个相框**不能**是普通的单色边框。必须使用 CSS 渐变边框（border-image 配合 linear-gradient），颜色从浅蓝色（#a2c2e0 类似色）过渡到浅水绿色（#a8d5ba 类似色），并带有一点类似水彩笔触的轻微不规则感（可以通过微调 box-shadow 或双层 div 嵌套实现渐变背景+内部白底的相框效果）。相框内部放置贝壳图片，底部是加粗的名称和灰色的简短描述。

* **右侧卡片：3D Gallery（重点：3D渐变气泡）**
    * 标题居中。
    * 包含两个并排的项目（Murex 和 Scallop）。
    * **气泡定制 CSS 需求**：这是设计的灵魂。不要用方形相框，图片要放在一个完美的圆形中（rounded-full）。
    * 必须使用复杂的 CSS 来模拟透明气泡：
        1.  `background`: 极度微弱的径向渐变（radial-gradient），中心几乎透明，边缘带有淡淡的浅紫/浅蓝色。
        2.  `box-shadow`: 必须包含多层阴影来实现立体感。包括内部的高光（inset 白色阴影在左上角），内部的暗部（inset 浅蓝色阴影在右下角），以及外部的柔和发光。
        3.  `backdrop-filter`: 适度的模糊效果。
    * 在主气泡周围，请用绝对定位（absolute）随机添加 2-3 个微小的半透明渐变小圆圈，模拟周围的小气泡。
    * 气泡下方带有小型的“View Model”渐变胶囊按钮。
